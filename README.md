# Wine Quality Prediction using Machine Learning

This project was developed as part of a Machine Learning workshop.
The project predicts wine quality using machine learning models trained on red wine and white wine datasets.

## Technologies Used
- Python
- NumPy
- Keras
- Machine Learning

## Project Files
- redwine_quality_training.py
- redwine_quality_predict.py
- winequality_white_training.py
- winequality_white_predict.py
- winequality-red.csv
- winequality-white.csv
- redwine_model.keras
- whitewine_model.keras

## Features
- Red wine quality prediction
- White wine quality prediction
- Training and prediction modules
- Dataset-based analysis

## Repository Contents
This repository contains datasets, training scripts, prediction scripts, and trained model files used for the project.
